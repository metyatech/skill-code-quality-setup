# Contributing

Thank you for your interest in contributing to `skill-code-quality-setup`.

## Development setup

```bash
git clone https://github.com/metyatech/skill-code-quality-setup.git
cd skill-code-quality-setup
```

## Submitting changes

1. Fork the repository and create a feature branch.
2. Add or update tests for any changed behavior.
3. Open a pull request with a clear description of the change.

## Scope

This skill covers **code quality toolchain setup** (formatters, linters, type checkers, dependency scanners). Please keep PRs scoped to tooling configuration and setup concerns.
